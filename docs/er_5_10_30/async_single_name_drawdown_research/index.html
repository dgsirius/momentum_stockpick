研究口径

全样本最大回撤引用了未来样本信息，只能作为研究上限，不是可直接执行参数。

  • 权重公式w_i = 8% + 5% * clip((dd_i/ref_i - start_frac)/(full_frac-start_frac), 0, 1)^gamma
  • 权重时点day t weight uses ticker drawdown through day t-1
  • 全样本警告full_sample uses future max drawdown over the full period; research-only
  • 费用口径no financing/transaction cost; local-currency returns
  • 最优参数start_frac=0.50，full_frac=0.80，gamma=6.00
